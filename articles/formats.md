### Formats Supported

<br/>

Bltiline supports the following formats:

<br/>

**Images:**
- jpg
- jpeg
- tiff
- tif
- png
- webp
- bmp

**Raw:**
- arw
- nef
- orf
- rw2

**Vector:**
- ai
- svg
- eps
- pdf

**Misc:**
- psd
- dicom
- cr2
- dcm
- dwg

<br/>

Blitline also supports certain video style formats for things such as video keyframes and gif-to-video conversion.

<br/>

**Special**
- gif
- avi
- mov
- mp4 
- mpeg
- mpg 
- wmv 
- 3g2 
- 3gp 
- asf
- flv 
- rm
- swf
- vob 
- m2v 
- m4v 
- webm
